# Código de Conducta

## Nuestro compromiso

Nos comprometemos a mantener un entorno abierto y acogedor para todos,
independientemente de edad, género, identidad, origen, orientación sexual
o creencias.

## Comportamiento esperado

- Mostrar respeto hacia los demás.
- Aceptar críticas constructivas.
- Utilizar lenguaje inclusivo.
- Colaborar de buena fe.

## Comportamiento inaceptable

- Ataques personales o insultos.
- Publicación de información privada.
- Comentarios despectivos o discriminatorios.
- Cualquier forma de acoso.

## Aplicación

En caso de violación del código, contacta con el profesorado o los responsables del proyecto.
Se revisará cada incidencia y se tomarán medidas adecuadas.
